Resumo

Node.js é uma framework open-source e que usa JavaScript no lado do servidor. A principal motivação do Node.js é a criação de aplicações de alta escalabilidade e com grande eficiência, sendo um dos grandes trunfos o uso de programação não-assíncrona.

O trabalho de casa baseou-se no desenvolvimento de um servidor web em Node.js relativo aos arqueossítios do Noroeste português. Foi também sugerida a adaptação das webpages sobre o mesmo tema geradas anteriormente para que pudessem ser servidas pela nossa aplicação.

Resumo

O trabalho prático desta semana tinha como objetivo a implementação de um servidor em Node.JS capaz de servir um website gerado com o dataset em formato JSON de um arquivo musical. Em primeiro lugar foi necessário gerar um ficheiro JSON para servir de índice ao nosso website, algo que foi alcançado usando um script em Python Para a geração do HTML dinamicamente foi usada a framework PUG.

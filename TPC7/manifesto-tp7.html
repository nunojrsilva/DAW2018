Resumo

O trabalho prático desta semana tinha como objetivo a implementação de um servidor em Node.JS capaz de servir um website para upload de ficheiros para um servidor. Devia ser possível fazer upload do ficheiro, indicando o respetivo ficheiro e uma descrição do seu conteúdo. Para garantir a persistência dos dados, é guardado um catálogo em ficheiro JSON com o nome do ficheiro, descrição, a sua localização e data de upload. Esse catálogo era apresentado aquando da submissão de um ficheiro e é possível servir estaticamente esse conteúdo. Para a geração dinâmica do HTML foi usada a framework PUG. Usamos tambem o Express para simplificar o código no servidor.
